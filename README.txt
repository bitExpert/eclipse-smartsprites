README
======

What is Eclipse Smartsprites?
----------------

Eclipse Smartsprites offers an integration of the SmartSprites engine into
the Eclipse IDE. The plugin allows you to easily maintain CSS sprites from
within your IDE.
The plugin contributes a context menu to the project explorer view. Right click
on any .css or .js file and choose External Tools > Generate Spritemap
